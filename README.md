# WorkGrid

WorkGrid is a multiproject task management app built with React + Firebase.

## Public Template Notes

This repository is sanitized for public sharing:

- No private task exports or personal data files are included.
- No local `.env` files are included.
- Firebase project configuration is read from environment variables.

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Create `.env.local` from `.env.example` and fill in your Firebase values.

3. Run locally:

```bash
npm run dev
```

4. Build:

```bash
npm run build
```
